# Hoster

Hoster is a modern website hosting platform where users can create, edit, and deploy websites directly from the browser.

## Features

* User accounts
* Live website editor
* HTML/CSS/JS editing
* Drag-and-drop builder
* Live preview
* Deploy system
* Custom domains
* Subdomain hosting
* AI website generator
* Analytics dashboard
* File uploads
* Dark/light themes

## Installation

```bash
npm install
node server.js
```

Open:

```text
http://localhost:3000
```
